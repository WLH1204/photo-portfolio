# 光影行迹 - 腾讯云部署指南

## 架构说明

```
┌──────────────┐     ┌──────────────────┐     ┌─────────────┐
│   Vercel     │────▶│  腾讯云 SCF       │────▶│  腾讯云 COS  │
│  (前端托管)   │     │  (云函数 API)     │     │  (存照片)    │
│  国内访问快   │     │  免费额度足够      │     │  国内速度快   │
└──────────────┘     └──────────────────┘     └─────────────┘
```

**免费额度**：
- COS：50GB 存储 + 50GB 流量/月（新用户）
- SCF：每月 100万次调用 + 40万 GBs 计算
- Vercel：无限静态托管

## 第一步：注册腾讯云

1. 访问 https://cloud.tencent.com/
2. 点击右上角「免费注册」
3. 用手机号注册（需要实名认证，但很快）
4. 新用户会送免费体验额度

## 第二步：创建 COS 存储桶

1. 登录腾讯云控制台
2. 搜索「对象存储 COS」
3. 点击「创建存储桶」
4. 设置：
   - **名称**：`photo-portfolio`（全局唯一）
   - **地域**：选择离你最近的（如广州 `ap-guangzhou`）
   - **访问权限**：**公有读私有写**
5. 点击「创建」

## 第三步：获取 API 密钥

1. 搜索「访问管理」→「API 密钥管理」
2. 点击「新建密钥」
3. 复制 **SecretId** 和 **SecretKey**（只显示一次！）

## 第四步：创建 SCF 云函数

1. 搜索「云函数 SCF」
2. 点击「新建」→「从头开始」
3. 设置：
   - **函数名称**：`photo-portfolio-api`
   - **运行环境**：Node.js 18.09
   - **地域**：和 COS 同一个地域（如广州）
4. 点击「下一步」
5. 上传代码：
   - 将 `tencent-api/` 文件夹中的所有文件打包成 zip
   - 点击「上传」→「从本地上传」
   - 选择 zip 文件
6. 点击「完成」

## 第五步：配置环境变量

1. 在云函数详情页 →「函数管理」→「函数配置」
2. 点击「编辑」→「环境变量」
3. 添加：

| 变量名 | 值 |
|--------|-----|
| `API_PASSWORD` | 你想设的密码 |
| `COS_BUCKET` | 你的桶名（如 photo-portfolio-1250000000） |
| `COS_REGION` | 你的地域（如 ap-guangzhou） |
| `COS_SECRET_ID` | 第三步的 SecretId |
| `COS_SECRET_KEY` | 第三步的 SecretKey |

4. 点击「保存」

## 第六步：创建函数 URL

1. 在云函数详情页 → 左侧菜单「**函数 URL**」
2. 点击「**创建**」
3. 设置：
   - **触发方式**：**公网访问** ✅
   - **认证方式**：**不需要认证**（我们自己有密码保护）
   - **请求方法**：**全部**
4. 点击「提交」
5. 创建成功后会显示**函数 URL**，类似：
   `https://photo-portfolio-api-xxx-1250000000.gz.app.myqcloud.com`
6. **复制这个地址**（不含末尾斜杠）

> 注意：在配置前端时，需要在地址后面加 `/api`，例如：
> `https://photo-portfolio-api-xxx-1250000000.gz.app.myqcloud.com/api`

## 第七步：部署前端到 Vercel

1. 安装 Vercel CLI：
```bash
npm i -g vercel
```

2. 登录 Vercel：
```bash
vercel login
```

3. 在项目根目录部署：
```bash
vercel --prod
```

4. 按提示操作：
   - Set up and deploy? → **Y**
   - Which scope? → 选择你的账户
   - Link to existing project? → **N**
   - Project name? → **photo-portfolio**
   - Directory where code is located? → **./**

5. 部署成功后会显示 Vercel 的 URL，类似：
   `https://photo-portfolio-xxx.vercel.app`

## 第八步：配置前端 API 地址

1. 打开你的 Vercel 网站
2. 点击右下角的 ⚙️ 齿轮按钮
3. 输入你的函数 URL 地址（第六步复制的地址 + `/api`）：
   例如：`https://photo-portfolio-api-xxx-1250000000.gz.app.myqcloud.com/api`
4. 点击「保存」
5. 输入你在第五步设置的密码
6. 点击「登录」

## 使用方式

### 你和女朋友的使用流程：

1. **你先配置好 API 地址**（第八步）
2. **登录**：在网站上输入你设置的访问密码
3. **开始使用**：上传照片、调整比例、换封面等操作会自动同步到云端
4. **分享给女朋友**：
   - 她打开同一个 Vercel 网址
   - 输入同一个访问密码
   - 她的操作也会同步到云端
   - 你们看到的是同一份数据

## 常见问题

### Q: 照片加载慢怎么办？
A: 确保 COS 和 SCF 在同一个地域（如都在广州），这样延迟最低。

### Q: 可以用自定义域名吗？
A: 可以，在 Vercel 绑定你的域名（需要 ICP 备案）。

### Q: 照片有大小限制吗？
A: 单张最大 30MB（COS 限制），但我们的代码会自动压缩到 800px。

### Q: 换了浏览器数据还在吗？
A: 在！只要登录同一个密码，数据会从云端同步下来。
